Flexbox

Flex box 設定分為兩部分:彈性容器(flex container)與彈性項目(flex items)。

彈性盒設定

1.display:flex

設定父元素 display:flex 為彈性容器，子元素們變成彈性項目(無論是甚麼類型的盒子，都會區塊化)。

彈性容器、彈性項目各自擁有屬性設定。

2.flex-direction 彈性盒軸向

彈性盒布局是以主軸和次軸為基礎，flex-direction 可以決定主軸水平或垂直，決定主軸方向後(次軸另一個方向)，知道主、次軸起點與終點，可以作為之後對齊的依據。

由於主軸方向是可變的，才會使用起點(start)、終點(end)，而不是左、右或上、下。

預設為 flex-direction: row。

2.flex-grow 彈性伸展係數

彈性項目的伸展係數，決定彈性項目是否伸展，預設 0 為不伸展

設成 1 或其他數字，會將剩餘的空間進行比例分配給彈性項目，例如三欄設定 1、2、3，剩餘空間比例分配為 1/6、6/2、3/6

3.flex-shrink 彈性收縮係數

彈性項目的收縮係數，預設 1，防止所有彈性項目加起來的總寬度超過彈性盒寬度而產生的爆版現象。

flex-shrink: 0，若彈性項目總寬度大於彈性盒寬度時，會產生爆版。
