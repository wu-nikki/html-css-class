01.Sass介紹

Sass 是 css的預處理器，目的在減少css的重工(重複使用，變成巢狀)

Sass 有變數、函式、流程控制、嵌套、繼承、mixins... 這些功能讓我們使用，比css在寫樣式時更有彈性。

Sass 可以更有效率維護樣式的結構

Sass需經過編譯成原生的css才能使用，瀏覽器看不懂Sass

02.Sass資料型態

  1. 1.number(數字):數字有分單位跟無單位，1,10,10px,20rem
  2. 2.string(字串):可用有引號字串或無引號字串
  3. 3.color(顏色):red,#fff,rgba(0,0,0,0.5)

  5. 5.null(空值):
  6. 6.list(像陣列):用半形空格或逗號作為分隔。例如:20px 0 20px，'Times New Roman', Times, serif'
  7. 7.map(物件):

03.Sass變數

宣告使用 $ ，變數位置即是作用域，就像JS的let宣告，寫在大括號裡面是區域變數，最外面是全域變數。

變數底線_ 和連結線- ，兩者是一樣的變數
.abc { $bg_123: red; $bg-123: blue; color: $bg_123; }
顯示出來的文字顏色為藍色

使用變數可以維持網站的一致性，節省修改成本，只需要修改變數，不需要在css修改。

    06.繼承

    繼承可以使用 css選擇器 或 % 佔位選擇器來命名

    使用css選擇器當作繼承，會被編譯到css。 %佔位符屬於scss的範圍，不會被編譯到css

    有共同樣式可使用繼承，會被聚集成群組選擇器，樣式碼會變少
