填充和對齊

1. nav-fill

使用 nav-fill 會將剩餘空間平均分給 nav-item 內容左右兩側空間。

.nav-fill 是 flex:1 1 auto = flex: auto = flex-grow: 1

  • 重量也要，普遍回來改變儘快一切都，接受。
  • Much longer nav link
  • Link
  • Disabled

2. nav-justified

想要等寬的 nav-item 使用 nav-justified

nav-justified 的 flex-basis 為 0，也就是讓寬度無效化，每個項目的分配空間會以 flex-grow 為主。

  • 重量也要，普遍回來改變儘快
  • Much longer nav link
  • Link
  • Disabled
